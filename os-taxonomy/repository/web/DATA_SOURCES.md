# 一至三年级数学数据来源与整理说明

本轮知识点来自用户提供的三份资料：

- `一年级代数知识点导图.pdf`
- `二年级知识点总结.pdf`

## 整理方法

导图中的一级、二级标题用于确定知识领域和学习阶段；能够独立教学、练习和验证的内容被整理为星系图中的微知识点。导图中的说明、例题和算法步骤被写入知识点描述或掌握表现，不单独膨胀为节点。

依赖边表示“学习后一个知识点前，通常需要先理解前一个知识点”，并综合考虑：

- PDF 导图的分支顺序
- 数概念到运算的认知顺序
- 口算到笔算、单步到混合运算的难度递进
- 一、二、三年级之间的跨年级衔接

## 当前主学习链

### 一年级

数数与一一对应 → 认识 0 至 10 → 加减法意义 → 5 以内加减 → 6 至 10 的分与合 → 20 以内进退位加减 → 100 以内数位与不进退位加减。

### 二年级

100 以内进退位加减 → 大数与估算 → 乘法意义 → 2 至 9 的乘法口诀 → 平均分与除法 → 用口诀求商 → 倍数关系与有余数除法初步。

### 三年级

混合运算 → 三位数连加连减 → 里程表应用 → 多位数乘除口算与笔算 → 小数的认识、比较和加减 → 含 0 的乘除、验算与乘除混合。
